## Описание программы

`Random Password Generator` — это удобное десктопное приложение для создания надежных случайных паролей. Оно позволяет гибко настраивать длину и состав пароля, а также автоматически сохраняет историю сгенерированных паролей, чтобы вы не потеряли важные данные.

### Основные возможности:

1.  **Гибкая настройка:** 
    *   Выбор длины пароля с помощью удобного ползунка (от 4 до 64 символов).
    *   Включение/отключение использования заглавных букв, строчных букв, цифр и специальных символов.
2.  **Генерация в один клик:** Быстрое создание пароля, который сразу отображается в текстовом поле для удобного копирования.
3.  **История паролей:** Программа автоматически запоминает все созданные пароли вместе с датой и временем генерации и отображает их в виде таблицы.
4.  **Локальное сохранение:** История паролей безопасно сохраняется на вашем компьютере в файле `password_history.json`.
5.  **Валидация:** Защита от ошибок ввода (проверка минимальной/максимальной длины и требование выбрать хотя бы один набор символов).

## Инструкция по запуску

Приложение использует только стандартные библиотеки Python, поэтому установка дополнительных модулей не требуется.

1.  Убедитесь, что на вашем компьютере установлен **Python 3**.
2.  Скачайте файлы проекта (или склонируйте репозиторий).
3.  Откройте терминал (или командную строку) в папке с файлом `password_generator.py`.
4.  Выполните команду:
    ```bash
    python password_generator.py
    ```

## Примеры использования и тесты

*   **Тест 1: Стандартный пароль.**
    Оставьте все настройки по умолчанию (длина 12, все типы символов включены). Нажмите "Сгенерировать". Проверьте, что пароль появился в поле и в таблице истории.
*   **Тест 2: Только цифры.**
    Снимите галочки с A-Z, a-z и спецсимволов. Оставьте только "0-9". Измените длину на 6 (например, для PIN-кода). Нажмите "Сгенерировать". Убедитесь, что пароль состоит только из цифр.
*   **Тест 3: Защита от пустого выбора (Валидация).**
    Снимите все галочки (ни один тип символов не выбран). Нажмите "Сгенерировать". Программа должна показать всплывающее окно с ошибкой "Выберите хотя бы один тип символов!".
*   **Тест 4: Проверка истории.**
    Закройте программу после создания нескольких паролей. Запустите её снова. Убедитесь, что таблица истории загрузилась корректно и содержит ранее созданные пароли.
